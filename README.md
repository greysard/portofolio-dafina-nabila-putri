# portofolio-dafina-nabila-putri
Website portofolio statis yang menampilkan proyek Data Analyst &amp; Data Scientist, berfokus pada analisis data, visualisasi, dan insight bisnis. Dibangun menggunakan HTML dan CSS dengan desain modern, responsif, dan tanpa JavaScript. Portofolio ini berisi ringkasan profil, skill, pengalaman, serta proyek berbasis dataset nyata.
